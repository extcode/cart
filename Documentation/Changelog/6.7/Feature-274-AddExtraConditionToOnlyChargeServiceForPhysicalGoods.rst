=============================================================================
Feature: #274 - Add Extra Condition to only Charge Service for Physical Goods
=============================================================================

See `Issue 274 <https://github.com/extcode/cart/issues/274>`__

Description
===========

The new condition allows you to scale the cost of a service based on the gross
price calculation of all physical products in the shopping cart.

More information about the flex price configuration can be found
in the :ref:`shipping_method_flex_price` documentation.
